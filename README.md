# WebCAD — 브라우저 기반 DXF 편집기

설치 없이 브라우저에서 바로 실행되는 2D CAD 프로그램입니다. AutoCAD 등과 호환되는
**DXF**(R12 ASCII) 파일을 생성·편집·열람할 수 있습니다.

> DWG는 Autodesk 독점 바이너리 포맷이라 브라우저에서 직접 다루기 어렵습니다.
> DXF는 같은 도면 데이터를 담는 공개 교환 포맷으로, AutoCAD에서 `DWG ↔ DXF`로
> 서로 변환할 수 있습니다. (AutoCAD: `다른 이름으로 저장 → DXF`, 또는 무료
> `ODA File Converter`로 일괄 변환)

## 실행 방법
1. `index.html`을 더블클릭하면 브라우저에서 바로 열립니다.
   (또는 폴더에서 `npx http-server` 후 `http://localhost:8080` 접속)
2. 그 외 빌드/설치 과정은 없습니다.
3. **웹에 올려서 어디서나 접속**: 정적 파일(`index.html`, `cad.js`)이라 GitHub Pages /
   Netlify / Vercel 같은 무료 정적 호스팅에 그대로 올리면 URL로 접속됩니다.

## 모바일/태블릿 (터치) 조작
데스크톱(마우스)과 함께 터치 입력도 지원합니다:
- **한 손가락 탭** = 클릭(점 찍기·선택)
- **선택 도구에서 한 손가락 드래그** = 박스 선택 (왼→오 윈도우 / 오→왼 크로싱)
- **그 외 도구에서 한 손가락 드래그** = 화면 이동(팬)
- **두 손가락** = 확대/축소(핀치) + 이동 (어느 도구에서나)
- **두 손가락 탭** = 우클릭(폴리라인 완료 / 작도 취소 / 선택 도구로 복귀)

### 저장 (모바일)
- **DXF 저장** 버튼 → 공유 시트가 뜨면 **"파일에 저장"** 으로 저장됩니다(iOS/Android).
- 작업물은 **자동으로 브라우저에 저장**되어, 실수로 새로고침/탭 종료해도 **다시 열면 복원**됩니다.
  완전히 새로 시작하려면 **"새로 만들기"**.

### 전체화면 (특히 iPhone)
iPhone Safari는 웹페이지 요소 전체화면을 지원하지 않습니다. 대신:
- Safari **공유 → "홈 화면에 추가"** → 홈 화면 아이콘으로 실행하면 **브라우저 UI 없이 전체화면**(standalone)으로 열립니다.
- 아이패드/안드로이드는 상단 **전체화면** 버튼도 동작합니다.

> 화면이 좁은 휴대폰에서는 좌측 도구막대·우측 패널이 캔버스를 많이 차지합니다.
> 태블릿이나 가로 모드에서 더 쾌적합니다.

## 기능
- **그리기**: 선, 폴리라인, 사각형, 원, 호, 문자
- **수정 명령(CAD)**: `offset`(간격복사), `copy`(복사), `mirror`(대칭),
  `array`(배열: 직사각형/원형), `rotate`(회전)
- **편집**: 선택(클릭·영역 드래그), 이동, 그립 편집, 속성 수정, 삭제
- **명령행**: 하단 입력창에 명령어를 타이핑(스페이스바로 포커스). 실제 CAD처럼 작동
- **레이어**: 추가/삭제/이름변경, 색상, 표시 토글
- **뷰**: 마우스 휠 줌, 가운데버튼(또는 Alt+드래그) 팬, 전체보기(F)
- **그리드 / 스냅**: 격자 표시 및 좌표 스냅
- **실행취소/다시실행** (Ctrl+Z / Ctrl+Y)
- **DXF 열기 / 저장**

## 명령 콘솔 (하단)
화면 하단에 **명령 기록(로그) + 입력창**으로 구성된 콘솔이 있습니다.
- **입력창**에 명령어를 타이핑하고 `Enter` 또는 `스페이스바`로 확정 (예: `offset`, `tr`, `fillet`).
- **빈 칸에서 `스페이스바`/`Enter` = 직전 명령 반복** (AutoCAD 관습). 캔버스에서 스페이스바를 눌러도 동일하게 반복됩니다.
- 숫자만 입력하면 진행 중 명령의 수치 인자가 됩니다 (오프셋 거리, 회전 각도, 모깎기 반지름, 배율).
- **로그 칸**에 입력한 명령(`명령: …`), 실행한 도구(`▶ …`), 결과(`✔ …`)가 순서대로 기록됩니다.

## 수정 명령 사용법
명령은 ① 도구 막대 버튼, ② 하단 **명령행에 이름 입력** 두 방법으로 실행합니다.
도형을 미리 선택하지 않았으면 첫 클릭으로 한 개를 집습니다.

| 명령 | 별칭 | 순서 |
|------|------|------|
| **offset** | o | 거리 입력(숫자 Enter) → 도형 클릭 → 오프셋 방향 클릭 (반복) |
| **copy** | co, cp | (선택) → 기준점 → 붙일 위치 클릭 (반복, Esc 종료) |
| **mirror** | mi | (선택) → 대칭축 두 점 클릭 (원본 유지) |
| **rotate** | ro | (선택) → 회전중심 → 각도 지점 클릭 *또는* 각도(°) 입력 |
| **array** | ar | (선택) → 설정창(직사각형 행·열·간격 / 원형 개수·각도) → (원형은) 중심 클릭 |
| **trim** | tr | **기준선 모드(기본)**: 기준 객체들 클릭 → Space 확정 → 그 기준선에 걸치는 부분만 잘림(AutoCAD 클래식). **기준 없이 바로 Space** → 빠른 모드(모든 객체가 경계, 클릭 즉시 자름). 우클릭=Space |
| **extend** | ex | 늘릴 선의 끝쪽 클릭 → 가장 가까운 경계까지 연장 |
| **fillet** | f | 반지름 입력(숫자 Enter) → 첫 선 → 둘째 선. 반지름 0이면 직각 모서리 |
| **scale** | sc | (선택) → 기준점 → 배율(숫자) 입력 *또는* 참조길이·새길이 두 점 클릭 |
| **stretch** | st | 걸침영역 두 모서리 → 기준점 → 이동점. 영역에 걸친 끝점만 이동 |
| **polygon** | pol | 변 개수(숫자) → 중심 → 반지름/꼭짓점. 정다각형 |
| **ellipse** | el | 중심 → 코너 클릭 또는 `rx,ry` 입력. 타원 |
| **chamfer** | cha | 거리(숫자) → 첫 선 → 둘째 선. 모따기 |
| **explode** | x | 선택한 폴리라인(사각형·다각형)을 선들로 분해 |
| **join** | j | 선택한 선/폴리라인을 이어 하나의 폴리라인으로 결합 |
| **dim** | dli, dal | 첫 점 → 둘째 점 → 치수선 위치. 치수 레이어에 치수 기입 |
| **dist** | di | 두 점 클릭 → 거리·ΔX·ΔY·각도 표시 |
| **area** | aa | 원/닫힌 폴리라인 클릭 또는 점들 지정 후 Enter → 면적·둘레 |
| **zoom** | z | 도면 전체보기 |
| **undo / redo** | u | 실행취소 / 다시실행 |
| **break** | br | 선/원/호 선택 → 끊기점 2개 → 사이 구간 제거 (원은 호로) |
| **lengthen** | len | 증감량(±) 입력 → 선의 끝쪽 클릭 → 길이 조정 |
| **hatch** | h | 경계(원/닫힌 폴리라인) 클릭 → **단일 해치 객체** 생성. 패턴 8종: ansi31(사선)·ansi37(격자사선)·steel(강재)·grid(타일)·brick(벽돌)·concrete(콘크리트)·dots(점)·solid(채움). 숫자=간격, 패턴명 입력으로 변경. 해치 클릭 후 속성 패널에서 패턴 교체 가능. DXF 저장 시 선으로 분해 |
| **dimradius / dimdiameter** | dra / ddi | 원·호 클릭 → 문자 위치 클릭 → R값/⌀값 치수 기입 |

> 실행 전 결과가 점선 고스트로 미리보기 됩니다. `Esc`로 명령 취소, `스페이스바`로 명령행 포커스.

## 단축키
| 키 | 기능 | 키 | 기능 |
|----|------|----|------|
| `L` | 선 | `T` | 문자 |
| `P` | 폴리라인 | `M` | 이동 |
| `R` | 사각형 | `F` | 전체보기 |
| `C` | 원 | `Del` | 삭제 |
| `A` | 호 | `Esc` | 선택도구/취소 |
| `Ctrl+Z` | 실행취소 | `Ctrl+S` | DXF 저장 |
| `Ctrl+Y` | 다시실행 | `Ctrl+A` | 전체 선택 |

## 작도 팁
- **선/사각형/원**: 두 번 클릭(시작 → 끝).
- **폴리라인**: 점을 연속 클릭, **우클릭 또는 Enter**로 완료.
- **호**: 중심 → 시작점 → 끝점 순서로 3번 클릭.
- 우클릭은 작도 취소/완료 및 선택도구 복귀로 동작합니다.

## 지원 DXF 엔티티
`LINE`, `LWPOLYLINE`, `CIRCLE`, `ARC`, `TEXT`(읽기 시 `MTEXT`도 단순 변환)
및 `LAYER` 테이블(색상/표시 상태).

## 파일 구성
- `index.html` — UI / 레이아웃
- `cad.js` — 캔버스 렌더링, 편집 로직, DXF 읽기/쓰기 (외부 의존성 없음)
- `sample.dxf` — 예제 도면 (DXF 열기로 불러와 보세요)
