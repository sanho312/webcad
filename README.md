# WebCAD — 브라우저 기반 DXF 편집기

설치 없이 브라우저에서 바로 실행되는 2D CAD 프로그램입니다. AutoCAD 등과 호환되는
**DXF**(R12 ASCII) 파일을 생성·편집·열람할 수 있습니다.

> DWG는 Autodesk 독점 바이너리 포맷이라 브라우저에서 직접 다루기 어렵습니다.
> DXF는 같은 도면 데이터를 담는 공개 교환 포맷으로, AutoCAD에서 `DWG ↔ DXF`로
> 서로 변환할 수 있습니다. (AutoCAD: `다른 이름으로 저장 → DXF`, 또는 무료
> `ODA File Converter`로 일괄 변환)

## 실행 방법
1. `index.html`을 더블클릭하면 브라우저에서 바로 열립니다.
   (또는 폴더에서 `npx http-server` 후 `http://localhost:8080` 접속)
2. 그 외 빌드/설치 과정은 없습니다.
3. **웹에 올려서 어디서나 접속**: 정적 파일(`index.html`, `cad.js`)이라 GitHub Pages /
   Netlify / Vercel 같은 무료 정적 호스팅에 그대로 올리면 URL로 접속됩니다.

## 모바일/태블릿 (터치) 조작
데스크톱(마우스)과 함께 터치 입력도 지원합니다:
- **한 손가락 탭** = 클릭(점 찍기·선택)
- **선택 도구에서 한 손가락 드래그** = 박스 선택 (왼→오 윈도우 / 오→왼 크로싱)
- **그 외 도구에서 한 손가락 드래그** = 화면 이동(팬)
- **두 손가락** = 확대/축소(핀치) + 이동 (어느 도구에서나)
- **두 손가락 탭** = 우클릭(폴리라인 완료 / 작도 취소 / 선택 도구로 복귀)

### 저장 (모바일)
- **DXF 저장** 버튼 → 공유 시트가 뜨면 **"파일에 저장"** 으로 저장됩니다(iOS/Android).
- 작업물은 **자동으로 브라우저에 저장**되어, 실수로 새로고침/탭 종료해도 **다시 열면 복원**됩니다.
  완전히 새로 시작하려면 **"새로 만들기"**.

### 전체화면 (특히 iPhone)
iPhone Safari는 웹페이지 요소 전체화면을 지원하지 않습니다. 대신:
- Safari **공유 → "홈 화면에 추가"** → 홈 화면 아이콘으로 실행하면 **브라우저 UI 없이 전체화면**(standalone)으로 열립니다.
- 아이패드/안드로이드는 상단 **전체화면** 버튼도 동작합니다.

> 화면이 좁은 휴대폰에서는 좌측 도구막대·우측 패널이 캔버스를 많이 차지합니다.
> 태블릿이나 가로 모드에서 더 쾌적합니다.

## 기능
- **그리기**: 선, 폴리라인, 사각형, 원, 호, 문자
- **수정 명령(CAD)**: `offset`(간격복사), `copy`(복사), `mirror`(대칭),
  `array`(배열: 직사각형/원형), `rotate`(회전)
- **편집**: 선택(클릭·영역 드래그), 이동, 그립 편집, 속성 수정, 삭제
- **명령행**: 하단 입력창에 명령어를 타이핑(스페이스바로 포커스). 실제 CAD처럼 작동
- **레이어**: 추가/삭제/이름변경, 색상, 표시 토글
- **뷰**: 마우스 휠 줌, 가운데버튼(또는 Alt+드래그) 팬, 전체보기(F)
- **그리드 / 스냅**: 격자 표시 및 좌표 스냅
- **실행취소/다시실행** (Ctrl+Z / Ctrl+Y)
- **DXF 열기 / 저장**

## 명령 콘솔 (하단)
화면 하단에 **명령 기록(로그) + 입력창**으로 구성된 콘솔이 있습니다.
- **입력창**에 명령어를 타이핑하고 `Enter` 또는 `스페이스바`로 확정 (예: `offset`, `tr`, `fillet`).
- **빈 칸에서 `스페이스바`/`Enter` = 직전 명령 반복** (AutoCAD 관습). 캔버스에서 스페이스바를 눌러도 동일하게 반복됩니다.
- 숫자만 입력하면 진행 중 명령의 수치 인자가 됩니다 (오프셋 거리, 회전 각도, 모깎기 반지름, 배율).
- **로그 칸**에 입력한 명령(`명령: …`), 실행한 도구(`▶ …`), 결과(`✔ …`)가 순서대로 기록됩니다.

## 작도 편의 (추가)
- **레이어 잠금(🔒)**: 레이어 헤더의 자물쇠로 잠그면 보이되 선택·수정 불가(밑그림·치수 보호). `👁 전체` = 모두 표시+잠금해제.
- **문자 더블클릭 편집**: 문자를 더블클릭하면 바로 내용 수정.
- **그리기 순서**: 선택 후 `front`/`back`(또는 속성 패널 버튼)으로 맨 앞/맨 뒤로. 해치·채움이 가릴 때.
- **유사 선택(similar / ss)**: 선택한 것과 같은 종류+레이어를 한 번에 전부 선택.
- **지시선(leader / le)**: 화살표 지점 → 문자 위치 클릭 → 문구. 주석용.
- **객체 스냅 추적**: 스냅점 위에 잠깐 머문 뒤 수평/수직으로 이동하면 주황 추적선이 나타나 그 점과 정렬된 위치를 잡아줍니다.
- **뷰 저장/이동/이전**: `vs 이름`=현재 화면 저장, `vg 이름`=이동, `vl`=목록, `zp`=이전 뷰로.
- **밑그림 이미지**: 파일 메뉴 → 🖼 밑그림 이미지 넣기. 스캔 도면·사진을 바닥에 깔고 위에 트레이싱('밑그림' 레이어 자동 잠금, DXF 내보내기에서는 제외).

## 치수·검토 (추가)
- **치수 설정(⚙ 옵션 → 📐 치수)**: 치수 문자 높이(0=그리기설정 따름)·소수 자릿수(거리/면적에도 적용)·단위 표시(100mm) — 새 치수부터 반영.
- **중심선(centerline / cl)**: 원/호 클릭 → 십자 중심선('중심선' 레이어, 일점쇄선 자동).
- **구름마크(revcloud / rc)**: 두 코너 클릭 → 영역 둘레에 바깥으로 볼록한 호들(도면 검토 표시), 연속 기입.

## 작도·편집 확장 2차
- **도곽(frame)**: 배치점 클릭 → 용지(A4~A1)·축척 입력 → 용지 외곽+내부 테두리+표제란(제목/축척/날짜) 자동 생성('도곽' 레이어).
- **정렬(align / al)**: 선택 → 원본1점→목표1점→원본2점→목표2점 — 이동+회전(길이가 다르면 배율 적용 여부 확인).
- **레이어 격리(isolate / iso)**: 선택 도형의 레이어만 남기고 모두 숨김. `uniso`로 해제.
- **길이·면적 합계(sum)**: 선택한 도형들의 총 길이·총 면적을 명령 기록에 출력(자재 산출용).
- **구성선(xline / xl)**: 두 점 → 사실상 무한한 보조선(보조선 레이어) — 자르기 기준·스냅용.
- **한 점 끊기(breakpt / bp)**: 선/원/호를 지정한 지점에서 둘로 분리.

## BIM (2D → 3D, 1단계: 속성)
2D 도형에 입체 속성을 부여하는 BIM-lite 기능의 1단계입니다.
- **wall**: 선/폴리라인 선택 후 → 높이·두께 입력 = 벽. 평면에 두께 밴드로 표시.
- **slab**: 닫힌 폴리라인/원 → 두께 = 바닥판. / **column(col)**: 원/닫힌 폴리라인 → 높이 = 기둥.
- **door / window(win)**: 벽 선의 원하는 위치 클릭 → 폭 입력 = 개구부(문/창). 벽 밴드가 끊겨 보이고 창은 이중선.
- 속성 패널에서 높이·두께·씰 등 수정, **bimclear**로 해제. 클라우드 저장·실시간 협업에 자동 포함(DXF에는 저장 안 됨).
- **3d**: 벽·슬래브·기둥·개구부를 입체로 표시 — 드래그 회전·휠 줌·Shift드래그 이동, 평면/아이소 프리셋. 문은 뚫린 구멍, 창은 인방·창턱·반투명 유리로 자동 분할(외부 라이브러리 없음).
- **section(sec)**: 평면에 절단선 두 점 → 바라볼 방향 클릭 → 투영 깊이 입력 → **새 탭에 단면도 자동 생성** (잘린 벽·슬래브는 해치, 뒤 요소는 투영선, 창은 유리 표시, 지반선 포함). 생성된 도면은 일반 2D — 치수·수정·DXF 저장 전부 가능.
- **elevation(ev)**: 건물 밖에 기준선 → 건물 쪽 클릭 → **입면도 자동 생성** (창 위치·크기 정확 반영).
- **다층(Level)**: 그리기 설정 패널의 **층 선택 + ＋버튼**으로 층 추가(이름·레벨mm). 새 도형·BIM은 현재 층에 귀속되고 **다른 층은 흐리게 참조 표시**(체크 해제 가능). BIM 높이(base)가 층 레벨에 자동 정렬 → 3D는 층이 쌓여 보이고, 단면은 복층이 함께 잘립니다. 층 구성은 도면(탭)마다 독립 저장·클라우드 포함.

## 클라우드 편의 (추가)
- **Ctrl+S**: 클라우드 도면이 열려 있으면 클라우드로 저장(DXF 내보내기는 파일 메뉴).
- 파일명 옆 **●** = 클라우드 미저장 변경 표시. 창을 숨기거나 닫기 직전 자동 저장.
- 클라우드 도면을 열면 기기 화면 크기에 맞게 자동 전체보기.

## 다중 문서 탭
최상단 탭 바에서 브라우저처럼 **여러 도면을 동시에** 열고 전환합니다(+ 버튼=새 탭, ×=닫기). 각 탭은 엔티티·레이어·블록·뷰·실행취소 이력·파일명이 독립적이며, **Ctrl+C → 다른 탭 → Ctrl+V**로 탭 간 객체 복사가 됩니다. 탭 구성은 자동저장되어 새로고침 후에도 복원됩니다.

## 링크로 공유 (서버 불필요)
파일 메뉴 → **🔗 링크로 공유**: 현재 도면을 gzip 압축해 URL에 담습니다. 그 링크를 열면 도면이 그대로 복원됩니다(백엔드·로그인 없이 정적 호스팅만으로 동작). 링크는 클립보드에 복사됩니다. 큰 도면은 URL 길이 제한으로 잘릴 수 있어 그럴 땐 DXF로 공유하세요.

## 블록 (block / insert)
반복 요소(문·창·가구 등)를 **블록**으로 묶어 관리합니다. 도형 선택 후 `block`으로 정의(단일 객체화), `insert` 도구로 우측 목록에서 골라 **배율·회전** 지정해 배치. 블록은 하나의 객체처럼 이동/복사/회전되고, `explode`로 원래 도형으로 분해됩니다. DXF의 BLOCK/INSERT로 저장·복원되어 다른 CAD와 호환됩니다.

## 인쇄 / 플롯 (⚙ 옵션 → 인쇄)
정해진 **축척으로 PDF 출력**: 용지(A4/A3/A2/A1/Letter)·방향·축척(1:1~1:1000 또는 용지맞춤)·출력영역(전체/창 드래그)·제목블록을 지정해 벡터 PDF를 만듭니다. 예: `1:100`이면 도면 100mm가 종이 1mm로 정확히 출력됩니다(단위 설정 반영).

## 폴리라인 정점 편집 (그립)
폴리라인을 선택하면 정점(채운 사각) 사이에 **중점 그립(빈 사각)** 이 표시됩니다. 중점 그립을 클릭-드래그하면 **정점이 추가**되고, 정점 그립을 **더블클릭하면 삭제**됩니다.

## 속성 편집
- **속성 일치(matchprop / ma)**: 원본 도형 클릭 → 대상들 클릭 → 레이어·색·선종류·선굵기(문자 높이·해치 패턴 포함) 복사.
- **다중 선택 일괄 변경**: 여러 도형 선택 시 속성 패널에서 레이어·색·선종류를 한 번에 변경.

## 레이어 (선종류·선굵기)
각 레이어에 **선종류**(실선/파선/숨은선/중심선(일점쇄선)/가상선(이점쇄선)/점선)와 **선굵기**(mm)를 지정할 수 있습니다. 화면에 즉시 반영되고 DXF의 LTYPE 정의·레이어 lineweight로 저장/복원됩니다.

## 옵션 (⚙ 상단 메뉴)
- **⌨ 명령 단축키**: 모든 명령에 원하는 단축 입력 지정 (예: `qq` → 원). 기본 별칭과 병행 사용.
- **🧲 객체 스냅 설정**: 끝점/중점/중심/수직/근처를 개별 켬/끔 (F3=전체 토글).
- **📏 단위 설정**: mm/cm/m/in — 좌표 표시·거리/면적 결과·DXF $INSUNITS 반영 (수치 변환은 하지 않음).
- 설정은 브라우저에 저장되어 다음 접속에도 유지됩니다.

## 수정 명령 사용법
명령은 ① 도구 막대 버튼, ② 하단 **명령행에 이름 입력** 두 방법으로 실행합니다.
도형을 미리 선택하지 않았으면 첫 클릭으로 한 개를 집습니다.

| 명령 | 별칭 | 순서 |
|------|------|------|
| **offset** | o | 거리 입력(숫자 Enter) → 도형 클릭 → 오프셋 방향 클릭 (반복) |
| **copy** | co, cp | (선택) → 기준점 → 붙일 위치 클릭 (반복, Esc 종료) |
| **mirror** | mi | (선택) → 대칭축 두 점 클릭 (원본 유지) |
| **rotate** | ro | (선택) → 회전중심 → 각도 지점 클릭 *또는* 각도(°) 입력 |
| **array** | ar | (선택) → 설정창(직사각형 행·열·간격 / 원형 개수·각도) → (원형은) 중심 클릭 |
| **trim** | tr | **기준선 모드(기본)**: 기준 객체들 클릭 → Space 확정 → 그 기준선에 걸치는 부분만 잘림(AutoCAD 클래식). **기준 없이 바로 Space** → 빠른 모드(모든 객체가 경계, 클릭 즉시 자름). 우클릭=Space |
| **extend** | ex | 늘릴 선의 끝쪽 클릭 → 가장 가까운 경계까지 연장 |
| **fillet** | f | 반지름 입력(숫자 Enter) → 첫 선 → 둘째 선. 반지름 0이면 직각 모서리 |
| **scale** | sc | (선택) → 기준점 → 배율(숫자) 입력 *또는* 참조길이·새길이 두 점 클릭 |
| **stretch** | st | 걸침영역 두 모서리 → 기준점 → 이동점. 영역에 걸친 끝점만 이동 |
| **polygon** | pol | 변 개수(숫자) → 중심 → 반지름/꼭짓점. 정다각형 |
| **ellipse** | el | 중심 → 코너 클릭 또는 `rx,ry` 입력. 타원 |
| **chamfer** | cha | 거리(숫자) → 첫 선 → 둘째 선. 모따기 |
| **explode** | x | 선택한 폴리라인(사각형·다각형)을 선들로 분해 |
| **join** | j | 선택한 선/폴리라인을 이어 하나의 폴리라인으로 결합 |
| **dim** | dli, dal | 첫 점 → 둘째 점 → 치수선 위치 → 이후 **클릭만으로 연속(체인) 기입**(같은 치수선 오프셋 유지, Esc 종료) |
| **dist** | di | 두 점 클릭 → 거리·ΔX·ΔY·각도 표시 |
| **area** | aa | 원/닫힌 폴리라인 클릭 또는 점들 지정 후 Enter → 면적·둘레 |
| **zoom** | z | 도면 전체보기 |
| **undo / redo** | u | 실행취소 / 다시실행 |
| **break** | br | 선/원/호 선택 → 끊기점 2개 → 사이 구간 제거 (원은 호로) |
| **lengthen** | len | 증감량(±) 입력 → 선의 끝쪽 클릭 → 길이 조정 |
| **hatch** | h | 경계(원/닫힌 폴리라인) 클릭 → **단일 해치 객체** 생성. 패턴 8종: ansi31(사선)·ansi37(격자사선)·steel(강재)·grid(타일)·brick(벽돌)·concrete(콘크리트)·dots(점)·solid(채움). 숫자=간격, 패턴명 입력으로 변경. 해치 클릭 후 속성 패널에서 패턴 교체 가능. DXF 저장 시 선으로 분해 |
| **dimradius / dimdiameter** | dra / ddi | 원·호 클릭 → 문자 위치 클릭 → R값/⌀값 치수 기입 |
| **dimangular** | dan | 두 선 클릭 → 호 위치 클릭 → 각도(°) 치수 |
| **divide / measure** | div / me | 개수·간격 입력 후 대상 클릭 → 등분점/일정간격 ✕ 표식 |
| **Ctrl+C / Ctrl+V** | — | 선택 복사 → 고스트 미리보기 → 클릭 배치 (블록 정의 동봉, 탭 간 공유) |

> 실행 전 결과가 점선 고스트로 미리보기 됩니다. `Esc`로 명령 취소, `스페이스바`로 명령행 포커스.

## 작도 팁
- **선/사각형/원**: 두 번 클릭(시작 → 끝).
- **폴리라인**: 점을 연속 클릭, **우클릭 또는 Enter**로 완료.
- **호**: 중심 → 시작점 → 끝점 순서로 3번 클릭.
- 우클릭은 작도 취소/완료 및 선택도구 복귀로 동작합니다.

## 로그인 / 회원가입 (이메일 인증)
`auth-config.js` 에 Supabase 프로젝트의 URL/anon key를 넣으면 로그인 게이트가 활성화됩니다
(비어 있으면 로그인 없이 기존처럼 동작).

1. https://supabase.com → **New project** 생성 (무료)
2. **Project Settings → API**: Project URL / anon public key 를 `auth-config.js` 에 붙여넣기
3. **SQL Editor** 에서 `supabase-setup.sql` 실행 (프로필 테이블 + 아이디 로그인용 함수)
4. **Authentication → Email Templates**: "Confirm signup"과 "Reset password" 본문에
   `{{ .Token }}` (6자리 인증번호)이 보이도록 수정. 예: `<h2>{{ .Token }}</h2>`
5. 커밋/푸시하면 끝. 회원가입 → 이메일 인증번호 입력 → 로그인 흐름이 동작합니다.

- 아이디 또는 이메일 + 비밀번호로 로그인, 비밀번호 재설정(인증번호) 지원.
- 무료 티어 기본 메일은 시간당 발송 제한이 작으므로, 실사용 시
  **Authentication → SMTP Settings** 에 커스텀 SMTP(예: Resend 무료) 연결 권장.
- 서버 없이 UI만 시험: 콘솔에서 `localStorage.setItem('webcad_auth_demo','1')` 후 새로고침
  (인증번호가 화면에 표시되는 데모 모드).

## 클라우드 기능 (로그인 계정 연동)
`supabase-cloud.sql`을 Supabase SQL Editor에서 실행하면 활성화됩니다.
- **☁ 클라우드 저장 / 내 도면**(파일 메뉴): 도면을 계정에 저장, 썸네일 목록에서 열기/이름변경/삭제. 어느 기기에서나 로그인하면 접근. 클라우드 도면은 변경 시 2분마다 자동 저장. 무료 플랜 5개(profiles.plan='pro'로 바꾸면 무제한).
- **🕘 버전 기록**: 저장마다 스냅샷(최근 10개) — 이전 버전 열어 복원.
- **👥 공유**: 상대 아이디로 읽기/편집 권한 공유. 공유받은 도면은 "내 도면" 목록에 👥 표시.
- **★ 블록 라이브러리**(옵션 메뉴): 도면의 블록을 계정 라이브러리에 올리고 어느 도면에서나 가져오기.
- **설정 동기화**: 단축키·단위·스냅 설정이 계정을 따라다님.
- **💬 피드백**(옵션 메뉴): 사용자가 앱에서 바로 버그/제안 전송 → feedback 테이블.
- **운영자용**: error_logs(자동 오류 수집), usage_stats(일자·기능별 사용 집계), announcements(행 추가하면 사용자 화면에 공지 배너), profiles.plan(free/pro 등급).
